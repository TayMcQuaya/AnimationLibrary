How to Use:

  1. Download the required files:
    • animate.min.css
    • animations.css
  2. Include the files in your HTML: 
    <link rel="stylesheet" href="path-to/animate.min.css">
                
<link rel="stylesheet" href="path-to/animations.css">
  3. Apply animation classes: Use the animation class in your HTML elements like so: 
    An example with Rotate 3D:
<div class="rotate3D">Your Content</div>

Make sure to use the correct class name and include both files for default and custom animations.
